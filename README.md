# Raipur Climate Trend Analysis (1951–2023)

## Overview
Analysis of 73 years of daily climate data for Raipur 
using Python to identify long-term warming trends, 
seasonal patterns, and extreme weather events.

## Key Findings
- Hottest month: May | Coldest month: January
- Peak rainfall year: 2021 (1,563 mm)
- Lowest rainfall year: 1997 (155 mm)
- Heatwave trends analyzed across 7 decades

## Tools Used
Python, Pandas, Matplotlib

## Dataset
26,800+ daily records covering temperature 
(max/min) and rainfall from 1951 to 2023
